Installation

ApPHP Framework can be installed on any modern hosting service that supports PHP or on a local web server. If you are going to install ApPHP on your local machine, and don't have a web server software, we recommend installing Zend CE with an all-in-one package.

Also you may check if your current server settings meet ApPHP Framework requirements here.
  1. Download ApPHP Framework: http://apphp.com/php-framework/.
  2. Extract the package contents on your computer and upload its contents to the remote server's directory via FTP, where PHP scripts should be installed (for local installation copy all files from the archive to htdocs folder on your Zend CE installation). The contents of the installation directory must be accessible via HTTP, for example: http://www.yourdomain.com/{installation_folder}/
  3. That's all! Congratulations, you have now ApPHP Framework installed.
